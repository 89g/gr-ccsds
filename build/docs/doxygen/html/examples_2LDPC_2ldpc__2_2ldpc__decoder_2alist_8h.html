This class handles sparse matrices specified in alist-format. For details about alist format please visit the link below.

  • http://www.inference.phy.cam.ac.uk/mackay/codes/alist.html

Alist class is an efficient way of representing a sparse matrix the parity check matrix H of an LDPC code for instance.
